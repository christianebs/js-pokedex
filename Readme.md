## 📝 Sobre o projeto

Este projeto é uma **Pokedex** que exibe todos os Pokemons da primeira geração. A solicitação é feita usando a fetch API e a API pokeapi.co.

## 📥 Tecnologias

 - HTML
 - CSS
 - JavaScript
 - Git & GitHub
